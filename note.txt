Defining transaction mongodb schema

Here’s a clean and practical example of how to define a Transaction schema using Mongoose — perfect for a loan or finance app.

🔍 Fields Explained:
Field	Description
user	Reference to the user making the transaction
loan	Reference to the related loan
amount	Amount involved in the transaction
type	Transaction type (payment, disbursement, etc.)
status	Status of the transaction
method	How the payment was made
transactionDate	When it happened
referenceId	Unique ID for tracking (e.g. from payment gateway)
