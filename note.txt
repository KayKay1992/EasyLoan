Auth Routes skeleton and defining

Sure! Here's a complete and clean Auth route setup using Express for your MERN loan app. This will include basic routes for:
User registration
User login
(Optional) Logout
(Optional) Get current user
✅ Get user profile (GET /profile)
🔄 Update user profile (PUT /profile)

router.post('/register', registerUser);
// 📝 Register a new user account

router.post('/login', loginUser);
// 🔑 Log in and receive a JWT token

router.get('/me', protect, getCurrentUser);
// 👤 Get basic info about the currently logged-in user

router.get('/profile', protect, getUserProfile);
// 📄 Fetch the full profile of the authenticated user

router.put('/profile', protect, updateUserProfile);
// ✏️ Update the authenticated user’s profile details
